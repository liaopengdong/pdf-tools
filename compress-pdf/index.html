Compress a PDF in the browser

This tool rewrites a PDF with object streams enabled. It can reduce some PDF files, especially files with inefficient structure, but it is not a full image recompression engine.

Best uses

  • Try to reduce eligible PDF files quickly.
  • Rewrite a PDF before uploading or sharing.
  • Optimize simple browser-readable PDFs without installing software.

FAQ

Will every PDF become smaller? No. If the source PDF is already optimized or mostly contains compressed images, the output may be similar in size or larger.

Does this reduce image quality? This browser tool does not aggressively recompress images. That keeps quality stable, but size reduction is limited.
