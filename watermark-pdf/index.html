Add a text watermark to PDF pages

This tool places a diagonal text watermark across each page of a PDF. It is useful for review copies, drafts, confidential documents, internal files, and sample documents.

Best uses

  • Mark a PDF as draft or confidential.
  • Add review status to shared documents.
  • Label sample documents before sending.

FAQ

Can I choose the watermark text? Yes. Enter your own text before processing.

Can this remove an existing watermark? No. This tool adds a new text watermark.
