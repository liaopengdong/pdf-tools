Extract text from a PDF

This converter reads selectable PDF text and saves it as a plain text file. It works best with digital PDFs where text can already be selected in a PDF viewer.

Important limitation

This is not OCR. Scanned image-only PDFs may return little or no text unless they already contain a text layer.

FAQ

Will formatting be preserved? No. The output is plain text.

Can it read scanned PDFs? Only if the scanned PDF already includes selectable text.
