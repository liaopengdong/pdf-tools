Extract only the PDF pages you need

Use this tool to create a smaller PDF from selected pages. It is useful when a large file contains only a few pages you need to share, upload, or print.

How page ranges work

  • Use 1-3 to extract pages 1 through 3.
  • Use 1-3, 7 to extract a range and a single page.
  • Leave the field empty to keep all pages.

FAQ

Does this change the original PDF? No. The tool creates a new PDF.

Can I extract non-consecutive pages? Yes, use comma-separated page numbers or ranges.
