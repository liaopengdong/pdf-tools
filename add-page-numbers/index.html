Add page numbers to a PDF

This tool adds page numbers in the format 1 / total to the bottom center of each PDF page. It is useful for review copies, document packets, and printed files.

Best uses

  • Number a report before sharing.
  • Prepare document packets for review.
  • Make printed PDFs easier to reference.

FAQ

Can I change the number style? This version uses a simple page / total format.

Will it overwrite existing numbers? It adds new numbers, so check documents that already have footers.
