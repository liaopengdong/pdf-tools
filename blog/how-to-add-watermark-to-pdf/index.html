How to add a watermark to a PDF

Watermarks help label documents as drafts, confidential files, samples, or review copies. They make the document status visible on every page.

Steps

  1. Open the watermark PDF tool.
  2. Select a PDF file.
  3. Enter the watermark text.
  4. Process and download the watermarked PDF.

Good watermark text

Common choices include Draft, Confidential, Sample, Internal Use Only, and Review Copy.
