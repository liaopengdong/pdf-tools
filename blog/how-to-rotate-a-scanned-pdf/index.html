How to rotate a scanned PDF

Scanned PDFs often contain pages that are sideways or upside down. Rotating the PDF updates how those pages display in PDF viewers.

Steps

  1. Open the rotate PDF tool.
  2. Select the PDF file.
  3. Choose 90, 180, or 270 degrees.
  4. Leave the range empty for all pages, or enter selected pages.

When to rotate selected pages

Use selected page ranges when only some pages are sideways in a mixed document.
