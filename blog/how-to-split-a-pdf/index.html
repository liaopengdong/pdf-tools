How to split a PDF into separate pages

Splitting a PDF creates individual page files. This helps when each page needs to be uploaded, reviewed, renamed, or sent separately.

Steps

  1. Open the split PDF tool.
  2. Select a PDF file.
  3. Download the ZIP file containing one PDF per page.

Split vs. extract

Use split when you want every page as a separate file. Use extract when you want only selected pages in one new PDF.
