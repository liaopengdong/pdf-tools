Reverse PDF page order

This tool changes page order from last-to-first. It is useful when scanned pages were saved backwards or when a document stack was fed into a scanner in reverse order.

Best uses

  • Fix backwards scanned packets.
  • Reverse exported page order.
  • Prepare documents for correct reading order.

FAQ

Does this rotate pages? No. It only changes page order.

Can I reverse only part of a PDF? This version reverses the full document.
