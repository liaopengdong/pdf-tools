Rotate PDF pages

Use this page when scanned documents, image-based PDFs, or mixed-orientation pages are sideways or upside down. You can rotate every page or enter a page range for selected pages.

Best uses

  • Fix sideways scanned pages.
  • Rotate landscape pages in a mixed PDF.
  • Correct upside-down pages before sharing.

FAQ

How do page ranges work? Use values like 1-3, 5. Leave the range empty to rotate all pages.

Does rotation change the page content? It updates PDF page rotation so viewers display the page in the chosen orientation.
